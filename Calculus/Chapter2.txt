导数的定义
1)设函数f(x)在点x0的某邻域内有定义，当x在x0处取得增量Δx时，
相应的函数f(x)取得增量Δy
如果Δy与Δx之比在Δx->0时的极限存在，则f(x)在x0处可导
f′(x0) = lim x->0 Δy / Δx = lim Δx->0 f(x0 + Δx) - f(x0) / Δx
= lim x->x0 f(x) - f(x0) / x - x0

2)若对于开区间I内的每一点x0，f(x)都可导，则
f'(x) = lim Δx -> 0 f(x + Δx) - f(x) / Δx

简单函数的导数
1)f(x) = c f'(x) = 0
2)f(x) = xⁿ f'(x) = nxⁿ⁻¹
3)f(x) = sinx f'(x) = cosx
4)f(x) = cosx f'(x) = +-sinx
5)f(x) = aˣ f'(x) = aˣlna
6)f(x) = loga x f'(x) = 1 / (x lna)
7)f(x) = |x| / x f'(x) 不存在

单侧导数
1)左导数
f'-(x0) = lim Δx -> 0⁻ f(x0 + Δx) / f(x0) / Δx
1)右导数
f'+(x0) = lim Δx -> 0⁺ f(x0 + Δx) / f(x0) / Δx

左右导数存在且相等 <=> f(x0)可导

导数的几何意义
曲线f(x)上某点M(x0, f(x0))的切线斜率为
 k = f'(x0)
切线方程 y - y0 = k (x - x0)
法线方程 y - y0 = -1/k (x - x0)

可导性与连续性的关系
若lim x->x0 f(x) = f(x0)
因为lim Δy / Δx = f'(x)
令lim α = 0
Δy / Δx = f'(x) + α
Δy = f'(x)Δx + αΔx
当x -> 0时，右式为0，即lim Δy = 0
lim [f(x) - f(x0)] = 0
所以根据f(x)在x0处可导能推出f(x)在x0连续
但是连续不能推出可导

四则运算后的函数的可导性
u,v为两个可导函数
(u ± v)' = u' ± v'
(u ± v ± w)' = u' + v' + w'
(uv)' = u'v + uv'
当v为常函数时 (uv)' = (Cu)' = Cu'
(uvw)' = u'vw + uv'w + uvw'
(u/v)' = (u'v - uv') / v²

反函数
已知函数y = f(x)单调，连续，则有其反函数x = f⁻¹(y)
(f⁻¹(y))' = lim Δx -> 0 Δx / Δy = lim Δy -> 0 1 / (Δy / Δx) = 1 / (f(x))'
由于连续性可知，Δy -> 0时，Δx -> 0

反函数的导数等于直接函数的导数的倒数

注意：若f⁻¹(y)是以y为自变量的函数，计算结果应通过y与x之间的关系，化为y的表达式

复合函数
y = f(g(x)) ，令y = f(u)，u = g(x)，且y在u处可导，u在x处可导
dy / dx = f'(u)g'(x) = (dy / du)(du / dx)
若y = f(g(k(x))) 则 dy / dx = (dy / du)(du / dv)(dv / dx)

lim Δu -> 0 Δy / Δu = f'(u)
令α是Δu -> 0时的无穷小
Δy / Δx = f'(u) + α
若Δu ≠ 0， Δy = f'(u)Δx + αΔx
若Δu = 0， α = 0，也满足上式
两边同除 Δx
Δy / Δx = f'(u)(Δu / Δx) + α(Δu /Δx)
当Δx -> 0时，因为u在x处可导，即
lim Δx -> 0 Δu / Δx = g'(x)
lim Δx -> 0 α = lim Δu -> 0 α = 0
所以
lim Δx -> 0 Δy / Δx = (f(g(x)))' = f'(u)g'(x)

二阶导数 f''(x) = (d / dx)(dy / dx) = (d²y / dx²)
二阶及二阶以上的导数统称高阶导数

显函数与隐函数
在某区间内，函数y = f(x)满足当x取任一值，y有唯一值与其对应，称函数f(x)是显函数
在某区间内，方程F(x, y) = 0满足当x取任一值，y有唯一值与其对应，称方程F(x, y)是隐函数
隐函数求导方法:左式，右式分别求导(在求高阶导时，结果中不要包含低阶导)

对于幂指函数y = uⁿ，若u，n都可导，可用对数求导法，化成lny = nlnu的形式在求导

设函数y = f(x)在某区间内有定义，x0及x0+Δx在这区间内，如果增量Δy = f(x0 + Δx) - f(x0)
可表示为Δy = AΔx + o(Δx)，其中A是不依赖Δx的常量，则称函数f(x)在点x0处可微，而AΔx叫做
函数y = f(x)在点x0处相应于自变量增量Δx的微分，记做dy，即
dy = AΔx

可微的定义dy相当于Δy = f(x0 + Δx) - f(x0)可化为Δy = AΔx + o(Δx)（A是不依赖Δx的常量）

函数可微的充要条件是函数可导
Δy = AΔx + o(Δx) -> Δy / Δx = A + o(Δx) / Δx
当Δx -> 0时 A = lim Δy / Δx = f'(x)
则可微就可导
lim Δy / Δx = f'(x) -> Δy / Δx = f'(x) + α(α是x -> 0时的无穷小)
Δy = f'(x)Δx + αΔx
当Δx -> 0时 aΔx = o(x) 则 Δy = f'(x)Δx + o(Δx)（f'(x)是不依赖Δx的常量）

当Δx -> 0时， lim Δx -> 0 Δy / dy = lim Δx -> 0 Δy / f'(x)Δx = 1 / f'(x) * lim x -> 0 Δy / Δx = 1
则Δx -> 0时， Δy与dy是等价无穷小，则Δy = dy + o(dy)
当f'(x) ≠ 0时，∆y = f(x + ∆x) - f(x)可用dy = f'(x)∆x表示，误差
是o(dy),在|∆x|极小时，∆y ≈ dy

通常称∆x是自变量的微分，称为dx，则∆x = dx
dy / dx = f'(x)

微分的几何意义
∆y是f(x)在点x0的增量，dy是f(x)在点x0的切线的增量，当|∆x|足够小时候，|∆x|远大于|∆y - dy|
所以可以近似的用dy代替∆y。也就是可以在几何上近似的用切线段代替曲线段。

dy = f'(x)dx
d(u + v) = (u' + v')dx = du + dv
d(u - v) = (u' - v')dx = du - dv
d(Cv) = Cdu
d(uv) = (u'v + uv')dx = vdu + udv
d(u / v) = [ (u'v - uv') / v^2 ]dx = (vdu - udv) / v^2
y = f(u), u = g(x)且可导， dy = y'dx = f'(u)g'(x)dx = f'(u)dg(x) = f'(u)du

因为dy ≈ ∆y => f'(x)∆x ≈ f(x0 + ∆x) - f(x0)
又因为x = x0 + ∆x，所以 f'(x0)(x - x0) ≈ f(x) - f(x0)
f(x) ≈ f'(x0)(x - x0) + f(x0)

误差：在实际测量过程中，难免测量值和实际值存在区别，称为误差
假设某个量的实际值是A，测量值是a，那么称|A - a|是绝对误差，|A - a|/|a|是相对误差
由于可以根据测量仪器的精度可以知道误差的范围，|A - a| ≤ δA，称δA是绝对误差限，δA/|a|是相对误差限
