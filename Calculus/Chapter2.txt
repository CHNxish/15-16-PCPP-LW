导数的定义
1)设函数f(x)在点x0的某邻域内有定义，当x在x0处取得增量Δx时，
相应的函数f(x)取得增量Δy
如果Δy与Δx之比在Δx->0时的极限存在，则f(x)在x0处可导
f′(x0) = lim x->0 Δy / Δx = lim Δx->0 f(x0 + Δx) - f(x0) / Δx
= lim x->x0 f(x) - f(x0) / x - x0

2)若对于开区间I内的每一点x0，f(x)都可导，则
f'(x) = lim Δx -> 0 f(x + Δx) - f(x) / Δx

简单函数的导数
1)f(x) = c f'(x) = 0
2)f(x) = xⁿ f'(x) = nxⁿ⁻¹
3)f(x) = sinx f'(x) = cosx
4)f(x) = cosx f'(x) = +-sinx
5)f(x) = aˣ f'(x) = aˣlna
6)f(x) = loga x f'(x) = 1 / (x lna)
7)f(x) = |x| / x f'(x) 不存在

单侧导数
1)左导数
f'-(x0) = lim Δx -> 0⁻ f(x0 + Δx) / f(x0) / Δx
1)右导数
f'+(x0) = lim Δx -> 0⁺ f(x0 + Δx) / f(x0) / Δx

左右导数存在且相等 <=> f(x0)可导

导数的几何意义
曲线f(x)上某点M(x0, f(x0))的切线斜率为
 k = f'(x0)
切线方程 y - y0 = k (x - x0)
法线方程 y - y0 = -1/k (x - x0)

可导性与连续性的关系
若lim x->x0 f(x) = f(x0)
因为lim Δy / Δx = f'(x)
令lim α = 0
Δy / Δx = f'(x) + α
Δy = f'(x)Δx + αΔx
当x -> 0时，右式为0，即lim Δy = 0
lim [f(x) - f(x0)] = 0
所以根据f(x)在x0处可导能推出f(x)在x0连续
但是连续不能推出可导

四则运算后的函数的可导性
(u ± v)' = u' ± v'
(u ± v ± w)' = u' + v' + w'
(uv)' = u'v + uv'
当v为常函数时 (uv)' = (Cu)' = Cu'
(uvw)' = u'vw + uv'w + uvw'
(u/v)' = (u'v - uv') / v²
