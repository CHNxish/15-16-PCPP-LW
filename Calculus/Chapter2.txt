导数的定义
1)设函数f(x)在点x0的某邻域内有定义，当x在x0处取得增量Δx时，
相应的函数f(x)取得增量Δy
如果Δy与Δx之比在Δx->0时的极限存在，则f(x)在x0处可导
f′(x0) = lim x->0 Δy / Δx = lim Δx->0 f(x0 + Δx) - f(x0) / Δx
= lim x->x0 f(x) - f(x0) / x - x0

2)若对于开区间I内的每一点x0，f(x)都可导，则
f'(x) = lim Δx -> 0 f(x + Δx) - f(x) / Δx

简单函数的导数
1)f(x) = c f'(x) = 0
2)f(x) = xⁿ f'(x) = nxⁿ⁻¹
3)f(x) = sinx f'(x) = cosx
4)f(x) = cosx f'(x) = +-sinx
5)f(x) = aˣ f'(x) = aˣlna
6)f(x) = loga x f'(x) = 1 / (x lna)
7)f(x) = |x| / x f'(x) 不存在

单侧导数
1)左导数
f'-(x0) = lim Δx -> 0⁻ f(x0 + Δx) / f(x0) / Δx
1)右导数
f'+(x0) = lim Δx -> 0⁺ f(x0 + Δx) / f(x0) / Δx

左右导数存在且相等 <=> f(x0)可导

导数的几何意义
曲线f(x)上某点M(x0, f(x0))的切线斜率为
 k = f'(x0)
切线方程 y - y0 = k (x - x0)
法线方程 y - y0 = -1/k (x - x0)

可导性与连续性的关系
若lim x->x0 f(x) = f(x0)
因为lim Δy / Δx = f'(x)
令lim α = 0
Δy / Δx = f'(x) + α
Δy = f'(x)Δx + αΔx
当x -> 0时，右式为0，即lim Δy = 0
lim [f(x) - f(x0)] = 0
所以根据f(x)在x0处可导能推出f(x)在x0连续
但是连续不能推出可导

四则运算后的函数的可导性
u,v为两个可导函数
(u ± v)' = u' ± v'
(u ± v ± w)' = u' + v' + w'
(uv)' = u'v + uv'
当v为常函数时 (uv)' = (Cu)' = Cu'
(uvw)' = u'vw + uv'w + uvw'
(u/v)' = (u'v - uv') / v²

反函数
已知函数y = f(x)单调，连续，则有其反函数x = f⁻¹(y)
(f⁻¹(y))' = lim Δx -> 0 Δx / Δy = lim Δy -> 0 1 / (Δy / Δx) = 1 / (f(x))'
由于连续性可知，Δy -> 0时，Δx -> 0

反函数的导数等于直接函数的导数的倒数

注意：若f⁻¹(y)是以y为自变量的函数，计算结果应通过y与x之间的关系，化为y的表达式

复合函数
y = f(g(x)) ，令y = f(u)，u = g(x)，且y在u处可导，u在x处可导
dy / dx = f'(u)g'(x) = (dy / du)(du / dx)
若y = f(g(k(x))) 则 dy / dx = (dy / du)(du / dv)(dv / dx)

lim Δu -> 0 Δy / Δu = f'(u)
令α是Δu -> 0时的无穷小
Δy / Δx = f'(u) + α
若Δu ≠ 0， Δy = f'(u)Δx + αΔx
若Δu = 0， α = 0，也满足上式
两边同除 Δx
Δy / Δx = f'(u)(Δu / Δx) + α(Δu /Δx)
当Δx -> 0时，因为u在x处可导，即
lim Δx -> 0 Δu / Δx = g'(x)
lim Δx -> 0 α = lim Δu -> 0 α = 0
所以
lim Δx -> 0 Δy / Δx = (f(g(x)))' = f'(u)g'(x)

二阶导数 f''(x) = (d / dx)(dy / dx) = (d²y / dx²)
二阶及二阶以上的导数统称高阶导数

基本函数的n阶导数
(eˣ)⁽ⁿ⁾ = eˣ
(ln(1+x))⁽ⁿ⁾ = (-1)ⁿ⁻¹(n-1)!/(1+x)ⁿ
sinx⁽ⁿ⁾ = sin(x + n*π/2)
cosx⁽ⁿ⁾ = cos(x + n*π/2)
(xʸ)⁽ⁿ⁾ = y(y-1)(y-2)...(y-n+1)x⁽ʸ⁻ⁿ⁾
(xʸ)⁽ʸ⁾ = y!
(xʸ)⁽ʸ⁺¹⁾ = 0

四则运算后的函数的n阶导数
(u + v)⁽ⁿ⁾ = (u)⁽ⁿ⁾ + (v)⁽ⁿ⁾
二项式定理：
(u + v)ⁿ = (y=0,n)Σ(n,y)Cuⁿ⁻ʸvʸ
莱布尼茨公式：
(uv)⁽ⁿ⁾ = (y=0,n)Σ(n,y)Cu⁽ⁿ⁻ʸ⁾v⁽ʸ⁾

显函数与隐函数
在某区间内，函数y = f(x)满足当x取任一值，y有唯一值与其对应，称函数f(x)是显函数
在某区间内，方程F(x, y) = 0满足当x取任一值，y有唯一值与其对应，称方程F(x, y)是隐函数
隐函数求导方法:左式，右式分别求导(在求高阶导时，结果中不要包含低阶导)

对于幂指函数y = uⁿ，若u，n都可导，可用对数求导法，化成lny = nlnu的形式在求导
