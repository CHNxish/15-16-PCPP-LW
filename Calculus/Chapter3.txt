费马引理
f(x)在x0的某邻域U(x0)内有定义，且f(x)在x0处可导，如果对于任意的x 属于 U(x0),
都有f(x) ≤ f(0)或f(x) ≥ 0,那么f'(x0) = 0

通常称函数值为0的点称为函数的零点
通常称导数等于0的点称为函数的驻点

罗尔定理
如果函数f(x)满足
	1）在闭区间[a, b]内连续
	2）在开区间(a, b)内可导
	3）在两端点处函数值相等，f(a) = f(b)
	则在(a, b)内至少存在一点ξ(a < ξ < b)，使得f'(ξ) = 0

拉格朗日中值定理
如果函数f(x)满足
	1）在闭区间[a, b]内连续
	2）在开区间(a, b)内可导
	则在(a, b)内至少存在一点ξ(a < ξ < b)，使得f'(ξ) = f(b) - f(a) / b - a
证明：
假设f(x)在[a, b]内连续且在(a, b)内可导
则端点a, b的值分别是f(a), f(b)
过a, b得到一条线性函数L(x) = f(a) + [f(b) - f(a) / b - a] * (x - a)
那么在闭区间[a, b]范围内，在f(x)上取一点N，在L(x)上取一点M，
令线段NM的值为φ(x) = f(x) - L(x) = f(x) - f(a) - [f(b) - f(a) / b - a] * (x -a)
根据罗尔定理，φ(x)在闭区间[a, b]内连续，在(a, b)内可导，且φ(a) = φ(b) = 0
所以在(a, b)内存在一点ξ(a < ξ <b)，使得φ'(ξ) = 0
所以在(a, b)内存在一点ξ(a < ξ <b)，使得f'(ξ) = f(b) - f(a) / b -a

假设f(x)在[x, x + ∆x]内连续，(x, x + ∆x)内可导
那么存在一点x + θ∆x(0 < θ < 1)，使得f'(x + θ∆x) = f(x + ∆x) - f(x) / ∆x
∆y = f'(x + θ∆x)*∆x(0 < θ < 1) 称为有限增量定理

例题
求证：x / 1 + x < ln(1 + x) < x
证明：显然f(x) = ln(1 + x)在区间[a, b]上满足拉格朗日中值定理
所以存在一点ξ(0 < ξ < x)，使得f'(ξ) = f(x) - f(0) / x
因为f(0) = 0，则f(x) = f'(ξ) * x = x / 1 + ξ
又因为0 < ξ < x，所以x / 1 + x < ln(1 + x) < x

柯西中值定理
如果参数方程f(x)及F(x)满足：
	1）在闭区间[a, b]上连续
	2）在开区间(a, b)上可导
	3）对于任一x属于(a, b)，F’(x)≠0
	那么在(a, b)上至少有一点ξ，使得等式
	f’(ξ) / F’(ξ) = f(b) - f(a) / F(b) - F(a)
