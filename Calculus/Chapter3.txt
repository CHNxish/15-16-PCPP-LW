费马引理
f(x)在x0的某邻域U(x0)内有定义，且f(x)在x0处可导，如果对于任意的x 属于 U(x0),
都有f(x) ≤ f(x0)或f(x) ≥ f(x0),那么f'(x0) = 0

通常称函数值为0的点称为函数的零点
通常称导数等于0的点称为函数的驻点

罗尔定理
如果函数f(x)满足
	1）在闭区间[a, b]内连续
	2）在开区间(a, b)内可导
	3）在两端点处函数值相等，f(a) = f(b)
	则在(a, b)内至少存在一点ξ(a < ξ < b)，使得f'(ξ) = 0

拉格朗日中值定理
如果函数f(x)满足
	1）在闭区间[a, b]内连续
	2）在开区间(a, b)内可导
	则在(a, b)内至少存在一点ξ(a < ξ < b)，使得f'(ξ) = f(b) - f(a) / b - a
证明：
假设f(x)在[a, b]内连续且在(a, b)内可导
则端点a, b的值分别是f(a), f(b)
过a, b得到一条线性函数L(x) = f(a) + [f(b) - f(a) / b - a] * (x - a)
那么在闭区间[a, b]范围内，在f(x)上取一点N，在L(x)上取一点M，
令线段NM的值为φ(x) = f(x) - L(x) = f(x) - f(a) - [f(b) - f(a) / b - a] * (x -a)
根据罗尔定理，φ(x)在闭区间[a, b]内连续，在(a, b)内可导，且φ(a) = φ(b) = 0
所以在(a, b)内存在一点ξ(a < ξ <b)，使得φ'(ξ) = 0
所以在(a, b)内存在一点ξ(a < ξ <b)，使得f'(ξ) = f(b) - f(a) / b -a

假设f(x)在[x, x + ∆x]内连续，(x, x + ∆x)内可导
那么存在一点x + θ∆x(0 < θ < 1)，使得f'(x + θ∆x) = f(x + ∆x) - f(x) / ∆x
∆y = f'(x + θ∆x)*∆x(0 < θ < 1) 称为有限增量定理

例题
求证：x / 1 + x < ln(1 + x) < x
证明：显然f(x) = ln(1 + x)在区间[a, b]上满足拉格朗日中值定理
所以存在一点ξ(0 < ξ < x)，使得f'(ξ) = f(x) - f(0) / x
因为f(0) = 0，则f(x) = f'(ξ) * x = x / 1 + ξ
又因为0 < ξ < x，所以x / 1 + x < ln(1 + x) < x

柯西中值定理
如果参数方程f(x)及F(x)满足：
	1）在闭区间[a, b]上连续
	2）在开区间(a, b)上可导
	3）对于任一x属于(a, b)，F’(x)≠0
	那么在(a, b)上至少有一点ξ，使得等式
	f’(ξ) / F’(ξ) = f(b) - f(a) / F(b) - F(a)

洛必达法则解决未定式(0/0 ∞/∞)
1）当x -> a时，函数f(x)及F(x)都趋向于零或无穷
2）在点a的某去心邻域内，f'(x)及F'(x)都存在且F'(x) ≠ 0
3）lim x -> a f'(x) / F'(x)存在(或为无穷大)
那么 lim x -> a f(x) / F(x) = lim x -> a f'(x) / F'(x)

泰勒中值定理
如果函数f(x)在含有x0的某开区间(a, b)内具有直到(n+1)阶的导数，则对任一的x∈(a, b)，有
f(x) = f(x0) + f'(x0)(x-x0) + [f''(x0)/2!](x - x0)² + ... + [f⁽ⁿ⁾(x0)/n!](x-x0)ⁿ + Rn(x)
其中Rn(x) = [f⁽ⁿ⁺¹⁾(ξ)/(n+1)!](x-x0)⁽ⁿ⁺¹⁾

泰勒多项式
pn(x) = f(x0) + f'(x0)(x-x0) + [f''(x0)/2!](x - x0)² + ... + [f⁽ⁿ⁾(x0)/n!](x-x0)ⁿ
带拉格朗日余项的n阶泰勒公式
f(x) = f(x0) + f'(x0)(x-x0) + [f''(x0)/2!](x - x0)² + ... + [f⁽ⁿ⁾(x0)/n!](x-x0)ⁿ + Rn(x)
拉格朗日余项
Rn(x) = [f⁽ⁿ⁺¹⁾(ξ)/(n+1)!](x-x0)⁽ⁿ⁺¹⁾

pn(x)与f(x)之间的误差为|Rn(x)|，当|Rn(x)| = |[f⁽ⁿ⁺¹⁾(ξ)/(n+1)!](x-x0)⁽ⁿ⁺¹⁾| = [M / (n+1)!]|(x-x0)⁽ⁿ⁺¹⁾|
故lim x -> x0 Rn(x) / (x - x0)ⁿ = 0，即Rn(x) = o[(x - x0)ⁿ]

带佩亚诺余项的泰勒公式
f(x) = f(x0) + f'(x0)(x-x0) + [f''(x0)/2!](x - x0)² + ... + [f⁽ⁿ⁾(x0)/n!](x-x0)ⁿ + o[(x - x0)ⁿ]

当x0 = 0，则ξ = θx (0 < θ < 1)
带拉格朗日余项的麦克劳林公式
f(x) = f(0) + f'(0)x + [f''(0)/2!]x² + ... + [f⁽ⁿ⁾(0)/n!]xⁿ + [f⁽ⁿ⁺¹⁾(θx)/(n+1)!]x⁽ⁿ⁺¹⁾ (0 < θ < 1)
带佩亚诺余项的麦克劳林公式
f(x) = f(0) + f'(0)x + [f''(0)/2!]x² + ... + [f⁽ⁿ⁾(0)/n!]xⁿ + o(xⁿ)

f(x) ≈ f(0) + f'(0)x + [f''(0)/2!]x² + ... + [f⁽ⁿ⁾(0)/n!]xⁿ

基本函数的带拉格朗日余项的麦克劳林公式
f(x) = e^x      e^x = 1 + x + x^2/2! + ... + x^n/n! + e^θx/(n+1)!*x⁽ⁿ⁺¹⁾ (0 < θ < 1)
f(x) = sinx    sinx = x - x^3/3! + x^5/5! + ... + (-1)^m-1 * x^2m-1/(2m-1)! + R2m (n = 2m)
f(x) = cosx    cosx = 1 - 1/2!*x^2 + 1/4!*x4 + ... + (-1)^m*1/(2m)!*x^2m + R2m+1(x) (n = 2m)

函数的单调性和函数的凹凸性与函数的导数
若f(x)在[a, b]内连续，在(a, b)内可导
若在(a, b)内f'(x)>0，则f(x)单调递增
若在(a, b)内f'(x)<0，则f(x)单调递减

设f(x)在区间I上连续，若对I上任意两点x1，x2恒有
f(x1 + x2) < f(x1)+f(x2) / 2 那么f(x)在图像上是凹的
f(x1 + x2) > f(x1)+f(x2) / 2 那么f(x)在图像上是凸的

设f(x)在[a, b]上连续，在(a, b)上有一阶导数和二阶导数，那么
若在(a, b)内f''(x)>0，则f(x)在[a, b]上图象是凹的
若在(a, b)内f''(x)<0，则f(x)在[a, b]上图像是凸的

驻点：函数在某区间内可导，且f'(x)=0的点
拐点：函数的凹凸性发生变化的点，即f''(x)在这点前后符号发生改变

函数的极值和函数的导数
若f(x)在x0的某邻域内有定义，且对于任一的x属于x0的某邻域有
f(x) < f(x0) 或 f(x) > f(x0)
则f(x0)是f(x)的极大值(极小值)

若f(x)在x0某邻域内可导
若f(x)在x0处取得极值，则f'(x0) = 0（必要条件）

若f(x)在x0处连续，在f(x)在x0的某邻域内可导，若
1)若x∈(x0-δ, x0)时，f'(x)>0，且x∈(x0, x0+δ)时，f'(x)<0
f(x)在x0处取得最大值
2)若x∈(x0-δ, x0)时，f'(x)<0，且x∈(x0, x0+δ)时，f'(x)>0
f(x)在x0处取得最小值
3)若f(x)在邻域内符号不变，则没有极值

若f(x)在x0处具有二阶导数且f'(x0)=0，f''(x)≠0，那么
1)f''(x)<0时，函数f(x)在点x0处取得极大值
2)f''(x)>0时，函数f(x)在点x0处取得极小值

上面的公式其实可以说明一点：
当某函数有n+1阶导数时，若f(x)在n阶的导数为0，且n+1阶导数不为0，则n阶导数的前后异号

最大值最小值问题
函数在[a, b]上连续，在(a, b)上可导，那么函数的最值可以取得
1)极值点 2)不可导点 3)端点

利用导数的性质，知道函数的性态，画函数图像
1)确定函数的f(x)的定义域和函数的特性并求出f'(x)和f''(x)
2)求出一阶导数和二阶导数的所有零点，并求出函数的间断点和一阶导数和二阶导数不存在的点，吧函数的定义域划分成几个部分
3)确定这些部分中导数的符号，从而确定函数图形的升降和凹凸

5)算出导数的零点和不存在点的函数值，定出图像上相应的点

曲率
弧微分公式 ds = √(1 + y'^2) dx

设光滑曲线上有点m0,m,m'，m0m的弧长为s，mm'的弧长为∆s，m与x轴的夹角为α，m与m'的夹角为∆α
平均曲率 k平 = |∆α|/|∆s|
曲率 k = lim ∆x -> 0 |∆α|/|∆s|
对于直线来说，∆s不为0，∆α是0，所以曲率k = 0，也就是我直观上来看的“直线不弯曲”
对于圆来说，若∆α不为0，则中心角为∆α，所以 ∆α / ∆s = (∆s / a)/∆s = 1 / a
(中心角：圆心与m,m'的连线段的夹角，a是圆的半径)
所以圆上任意亮点的曲率都一样，是1/a，半径越小的曲率越大，也就是弯曲的越厉害

假设曲线的直角方程坐标是f(x)，且f(x)具有二阶导数，因为tan a = y
y'' = sec^2α*dα/dx
dα = (y'' / 1+tan^2α)dx
dα = (y'' / 1+y'^2)dx
由弧微分可知ds = √(1 + y'^2) dx
k = dα / ds = y'' / (1+y')^(3/2)
