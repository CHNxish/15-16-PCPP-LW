增量：当平面上一个质点从一点移动到另一个点，其坐标的纯改变或增量通过把终点坐标减去起点坐标而求得。
(x1,y1) -> (x2,y2)  delta x= x2 - x1  delta y= y2 -y1

斜率 直线的斜率 m= delta y / delta x
存在正斜率和负斜率，水平线的斜率为0，垂直线没有斜率

平行线：具有相同斜率的非垂直直线
垂直线：当非垂直线l1和l2相互垂直时，其斜率满足m1 * m2 = -1

直线的方程：
过点(a,b)，水平线的方程是y=b,垂直线的方程是x=a
点-斜式方程：
y=m(x - x1) + y1 （已经知道方程过的一点(x1,y1)和斜率m）
斜率-截距方程：
y=m(x - 0) + b = mx + b (b为Y-截距)
截距：非垂直直线与y的交点的纵坐标就是直线Y-截距，非垂直直线与x的交点的横坐标就是直线X-截距
一般线性方程：
Ax + By = C (A和B不全为0)

回归分析：拟合数据的特殊曲线的类型的过程称为回归分析，该曲线就是回归曲线
线性回归方程，最小二乘法：
回归分析四个步骤
1）图示数据（散点图）
2）求回归方程，对直线而言方程的形式是y = mx + b
3）把回归方程的图重叠在散点图上进行拟合
4）如果拟合满意，用回归方程来预测不在表中的x值和y值

任一四边形的边的中点依次连起来，得到的四边形是平行四边形
