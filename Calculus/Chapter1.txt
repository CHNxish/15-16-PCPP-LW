变化率和极限

x在[x1,x2]上的平均变化率：
delta y / delta x = f(x1) - f(x2) / x1 -x2  (x1 -x2 ≠ 0)
变化率在图像上为点p(x1,f(x1))与点Q(x2,f(x2))的连线
即PQ的割线，也是PQ的斜率

极限的正式定义：
设f(x)定义在x0的一个可能不包括x0的开区间上（U(x0,δ)可能去心）  //基本函数条件
对于x趋近于x0时f(x)趋近于L，并记作                          //结论
limx->x0 f(x) = L
对于任意的ε>0，存在δ>0，使得x∈{x|0<|x-x0|<δ}时，有         //依据
|f(x) - L| < ε

求极限和单侧极限

lim f(x) = L , lim g(x) = M
和法则     lim ( f(x) + g(x) ) = L + M
差法则     lim ( f(x) - g(x) ) = L - M
积法则     lim ( f(x) * g(x) ) = L * M
乘常数法则  lim ( k * f(x) ) = k * L（k是常数）
商法则     lim ( f(x) / g(x) ) = L / M （M ≠ 0）
幂法则     lim ( f(x) )ˣʸ = L ˣʸ  角标x是常数，y是常数的倒数，y的倒数不能为0


代数多项式表达式：
如果P(x) = anxⁿ + an-1xⁿ⁻¹ + ... + a0
那么lim x->c P(x) = P(c) = ancⁿ + an-1xⁿ⁻¹ + ... + a0

如果P(x)和Q(x)都是多项式且lim Q(x) ≠ 0，那么
lim x->c P(x) / Q(x) = P(c) / Q(c)

代数消除法：当lim Q(x) = 0时（即分母极限为0），则利用分子分母有理化，
已知定理和已知公式

三明治定理（夹逼）定理：
当f(x)在x0的某去心邻域内有定义，若g(x) ≤ f(x) ≤ g(x)
且 lim g(x) = lim h(x) = L，则lim f(x) = L

公式：
lim x->0 sinx / x = 1

单侧极限：
左侧极限：设f(x)定义在(c,a)上，c<a，如果当x在区间(c,a)内
趋于a时，f(x)任意接近的趋于L，那么我们就说f在a有左侧极限：
lim x->a⁻ f(x) = L
右侧极限：设f(x)定义在(a,b)上，a<b，如果当x在区间(a,b)内
趋于a时，f(x)任意接近的趋于L，那么我们就说f在a有右侧极限：
lim x->a⁺ f(x) = L

lim x->a f(x) = L ⇔ lim x->a⁻ f(x) = L & lim x->a⁺ f(x) = L

当x趋向于∞时的极限

当x趋于无穷时，f(x)有极限L并记作
lim x->∞ f(x) = L

当x->∞时有理函数的极限：
x趋于常量的所有法则，定理都有效

当x->∞时，确定有理函数的极限，可以分子，分母同时除以
分母中x的最高幂次，结果如何取决于多项式的次

可以通过研究x->∞时f(x)的性态来研究x->0时f(1 / x)的性态

渐近线

直线y = b是函数f(x)的水平渐近线，如果有
lim x->∞ f(x) = b
直线x = a是函数f(x)的垂直渐近线，如果有
lim x->a f(x) = ∞

无穷极限的正式定义

我们说x趋于x0时，f(x)趋于无穷，并记作
lim x->x0 f(x) = ∞
如果对任何正实数B存在δ>0，使一切满足
0<|x-x0|<δ的x，有f(x)>B

终极形态模型
f的终极性态模型，当且仅当 lim x->∞ f(x) / g(x) = 1
当x->-∞时，满足上式的g是f的左侧终极性态模型，当x->+∞时，为右侧
如果f和g满足上式时，说明x趋于无穷时，f(x) = g(x)

若f(x)中分子比分母的次大1，我们可以根据长除法，求的一线性函数，该
线性函数是一条斜渐近线，同时也是f(x)的终极性态模型

在一点的连续性

f(x)区间[a, b]上的连续性
内点：函数f(x)在其定义域内的内点c处是连续的
lim x->c f(x) = f(c)
端点：函数f(x)在其定义域的左端点a与右端点b是连续的
lim x->a⁺ f(x) = f(a) 或 lim x->b⁻ f(x) = f(b)

函数f(x)在x = c处连续，当且仅当
1)f(c)存在 2)lim x->c f(x)存在 3)lim x->c f(x) = f(c)

连续性的扩展：
基本函数在其定义域内一定连续
连续函数的代数组合：
如果函数f和g在x=c连续，那么
f+g f-g f*g k*f f/g(g≠0) f⁰g 在x=c处连续

连续函数的中间值定理：
在闭区间[a, b]上连续的函数一定能取到f(a)与f(b)之间每一个值
如果f(a)*f(b) < 0，则一定至少存在一个零点

斜率和切线
曲线y = f(x)在点P(x0, f(x0))的斜率是
m = lim h->0 (f(x0 + h) - f(x0)) / h

极限序列{an}收敛到数L，如果对于每个正数ε，都对应一个整数
N，使得对于所有n
        n>N => |an - L| <ε
记作lim n->∞ an = L 如果这样的L不存在，则说{an}发散
