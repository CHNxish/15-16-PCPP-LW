相似矩阵和二次型

1.内积
α = (a1, a2, ..., an)^T, β = (b1, b2, ..., bn)^T
(α, β) = a1b1 + a2b2 + ... + anbn = α^Tβ

性质
(α, β) = (β, α)
(kα, β) = (α, kβ) = k(α, β)
(α, β + r) = (α, β) + (α, r)
(α, α) ≥ 0，若(α, α) = 0，则α = 0
若(α, β) = 0，则称α, β正交
若(α, α) = 1，则α是单位向量

2.若n维向量α1, α2, ..., αn是两两正交非零向量组，则
α1, α2, ..., αn线性无关，反之不对

3.设e1, e2, ..., en是向量空间V的一个基，且e1, e2, ...,
en是两两正交的单位向量，称e1, e2, ..., en是单位向量基

4.斯密特正交化 -> 单位正交向量组
Bn = An - (An, B1)/(B1, B1) * B1 + ... + (An, Bn-1)/(Bn-1, Bn-1) * Bn-1
rn = (1 / |Bn|)Bn

5.正交矩阵 设n阶矩阵满足A^TA = E
n阶矩阵满足正交矩阵的充分必要条件是a1, a2, ..., an两两正交且是单位向量

设P是n阶正交矩阵，称线性变换Y=PX是正交变化，且|Y| = |X|
∵ |Y|^2 = Y^TY = (PX)^TPX = X^TP^TPX = |X|^2
又∵ |Y| ≥ 0 |X| ≥ 0
∴ |Y| = |X|

特征值和特征向量
设A是n阶矩阵，若存在向量X和实数λ，使得
      AX = λX 成立(X≠0)
称λ为特征值，X是特征向量

AX = λX => (A - λE)X = 0
∵ X ≠ 0 ∴ r(A - λE) < n ∴ |A - λE| = 0
将|A - λE|=0称为矩阵A的特征方程

特征值和特征向量的基本性质
性质1.设矩阵A的特征值为λ1, λ2, ..., λn，则
 1.λ1 + λ2 + ... + λn = a11 + a22 + ... + ann = tr(A)
 2.λ1λ2...λn = |A|
性质2.对于Aα = λα 
  若矩阵A可逆，A^-1α = (1/λ)α A*α = (|A|/λ)α
  若f(A)是A的矩阵多项式，则f(A)α = f(λ)α
性质3.λ1 ≠ λ2，α1, α2, ..., αs是λ1对应的线性无关向量组，β1, β2, ..., βt是λ2对应的线性无关向量组，则α1, α2, ..., αs, β1, β2, ..., βt线性无关

重要公式 A(α1, α2, ..., αn) = (α1, α2, ..., αn)[λ1        ]
                                              [   λ2     ]
                                              [      .   ]
                                              [        λn]

相似矩阵
定义1.设AB是n阶矩阵，若存在可逆矩阵P，使得P^-1AP = B，称AB相似

注
1.A与A相似
2.A与B相似，则B与A相似
3.A与B相似，B与C相似，则A与C相似

(AB)^T = B^TA^T (AB)^-1 = B^-1A^-1
|AB| = |A||B|

定理1.若A与B相似，则AB又相同的特征方程，从而有相同的特征值
证 若A与B相似，则存在方阵P，使得P^-1AP = B
  |B - λE| = |P^-1AP - λE| = |P^-1AP - λP^-1P|
  = |P^-1(A - λE)P| = |P^-1||A-λE||P| = |A-λE|
对于∀λ是A的特征值，同时也是B的特征值

注
1.A与B相似 |A| = |B| tr(A) = tr(B) r(A) = r(B)
2.A与B相似 |A-λE| = |B - λE|，反之不对

定义2.设A是n阶矩阵，若存在可逆矩阵P，使得P^-1AP = ∧，
称为矩阵A对角化

定理2.设A是n阶矩阵，则矩阵A可对角化的充分必要条件是A有n个线性无关的特征向量

实对角矩阵对角化
        
            特征值      λ1 ≠ λ2         对角化
任意方阵    实数，复数   特征向量线性无关   有条件

实对称矩阵  实数        特征向量正交      无条件 Q^TAT = λ

若A^T = A，Aα1 = λ1α1， Aα2 = λ2α2，λ1 ≠ λ2
λ1(α, β) = (λ1α, β) = (Aα1, β) = (Aα)^Tβ = α^TA^Tβ = λ2α^Tβ = λ2(α, β)
∴ (λ1 - λ2)(α, β) = 0
又∵ λ1 ≠ λ2
∴ (α, β) = 0

二次型及其标准化
定义1.含n个变量的二次齐次函数
f(x1, x2, ..., xn) = a11x1^2 + a22x2^2 + ... + annxn^2 + 2a12x1x2 + 2a13x1x3 + ... + 2an-1,nxn-1xn为二次型
  1.若二次型只有平方项交叉项，称其为标准二次型，否则称为非标准二次型
  2.若二次型标准型中系数只有1，-1，0，称其为规范二次型

令f(x1, x2, x3) = 2x1^2 - x2^2 - 2x1x2 + 3x3^2 + 4x2x3
含A = [2  -1 0] x = [x1]
      [-1 -1 2]     [x2]
      [0  2  3]     [x3]
X^TAX = f(x1, x2, x3)
称A为二次型的矩阵，A的秩为二次型的秩

对于A, B的n阶方程，存在可逆矩阵P
  P^-1AP = B <=> 相似
  P^TAP  = B <=> 合同

将二次型化为标准型
定理1.设f = X^TAX为二次型，则存在正交变换X=QY，使得f = λ1x1^2 + ... + λnxn^2，其中λ1, λ2, ..., λn为特征值
f = X^TAX = (QY)^TAQY = Y^TQ^TAQY = Y^T[λ1      ]Y
                                       [  λ2    ]
                                       [     .  ]
                                       [      λn]
= λ1y1^2 + ... + λnyn^2

配方法将二次型化为标准型
题 f(x1, x2, x3) = x1^2 + 2x1x2 - 2x1x3 - 4x3^2
解 f(x1, x2, x3) = x1^2 + 2(x2 - x3)x1 + (x2 - x3)^2 - (x2 - x3)^2 - 4x3^2 = (x1 + x2 - x3)^2 - (x2 - x3)^2 - 4x3^2
令y1 = x1 + x2 - x3
  y2 = x2 - x3
  y3 = x3
f(x1, x2, x3) = y1^2 - y2^2 - 4y3^2
x与y唯一对应

         二次型标准型              几何形态
正交变换  λ1y1^2 + ... + λnyn^2    |X| = |Y| 几何不变

配方法   不唯一                    可能变

正定二次型
定义1.设f = X^TAX，若对任意X ≠ 0，使得X^TAX > 0，
称二次型为正定二次型，其对应矩阵A称为正定矩阵

性质1.标准二次型f = λ1x1^2 + ... + λnxn^2，正定的充分必要条件是λi > 0

性质2.实二次型经过有限次线性变换后其正定型不变
