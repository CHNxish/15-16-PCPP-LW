将线性方程组化成阶梯形线性方程组

数域F中m*n个数字aij(i = 1,2,...,m; j = 1,2,...n)排成m行n列，并括以圆括弧的数表称为数域F上的m*n的矩阵，通常用大写字母记做A或Am*n，有时记做A = (aij)m*n
[a11 a12 ... a1n]
[a21 a22 ... a2n]
[... ...     ...]
[am1 am2 ... amn]
当aij ∈ R，A为实矩阵
当aij ∈ C，A为复矩阵
当任意的aij都为0，记做0
当m=n时，A是n阶矩阵
系数矩阵A，增广矩阵(A,b)

有解的方程组称为相容方程组 无解的方程组称为不相容方程组

矩阵的加法 数量乘法 乘法
1）矩阵相等
如果两个矩阵A=(aij)和B=(bij)的行数和列数分别相等，且各对应元素也相等，即aij=bij，则称A和B相等，记作A=B
2）矩阵相加 当矩阵是同型矩阵，即行数和列数相同时，才可以相加
A + B = (aij + bij) = [a11 + b11 a12 + b12 ... a1n + b1n]
                      [a21 + b21 a22 + b22 ... a2n + b2n]
                      [...       ...           ...      ]
                      [am1 + bm1 am2 + bm2 ... amn + bmn]
3）矩阵的数量乘法
kA = (kaij) = [ka11 ka12 ... ka1n]
              [ka21 ka22 ... ka2n]
              [...  ...      ... ]
              [kam1 kam2 ... kamn]
4）矩阵的乘法 设A时一个m*n的矩阵，B时一个n*s的矩阵
A = [a11 a12 ... a1n] B = [b11 b12 ... b1s]
    [a21 a22 ... a2n]     [b21 b22 ... b2s]
    [... ...     ...]     [... ...     ...]
    [am1 am2 ... amn]     [bn1 bn2 ... bns]
C = AB = (cij) = ai1b1j + ai2b2j + ... ainbnj = (k=1 n)∑aikbkj

矩阵和行列式的区别
矩阵是m*n的数表
行列式是n*n的值
当D=|A|时，若D=0且A≠0，则A为奇异矩阵，若D≠0，则A为非奇异矩阵
