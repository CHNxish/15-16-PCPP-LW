矩阵

A = [a11 a12 ... a1n], 记做 A = (aij)mxn
    [a21 a22 ... a2n]
    [... ...     ...]
    [am1 am2 ... amn]
aij全为0，称为零矩阵，A = 0
m = n，称为n阶方阵
方阵除了对角线外全为0，A为对角矩阵
若A, B矩阵，行数和列数相同，称A, B为同型矩阵
若A, B是同型矩阵且各对应元素全部相等,A = B

矩阵运算(三则运算)
1.矩阵的加，减法(同型矩阵)
A = [a11 a12 ... a1n] B = [b11 b12 ... b1n]
    [a21 a22 ... a2n]     [b21 b22 ... b2n]
    [... ...     ...]     [... ...     ...]
    [am1 am2 ... amn]     [bm1 bm2 ... bmn]
A ± B = [a11 ± b11 a12 ± b12 ... a1n ± b1n]
        [a21 ± b21 a22 ± b22 ... a2n ± b2n]
        [...       ...           ...      ]
        [am1 ± bm1 am2 ± bm2 ... amn ± bmn]

2.矩阵的乘法
kA = [ka11 ka12 ... ka1n] (k为系数)
     [ka21 ka22 ... ka2n]
     [...  ...      ... ]
     [kam1 kam2 ... kamn]

Amxn Bnxs 内标同可乘 外标决定型
A = [a11 a12 ... a1n] B = [b11 b12 ... b1s]
    [a21 a22 ... a2n]     [b21 b22 ... b2s]
    [... ...     ...]     [... ...     ...]
    [am1 am2 ... amn]     [bn1 bn2 ... bns]
C = AB = [c11 c12 ... c1s]
         [c21 c22 ... c2s]
         [... ...     ...]
         [cm1 cm2 ... cms]
cij = ai1 * b1j + ai2 * b2j + ... + ain * bnj (i = 1, 2, ..., m j = 1, 2, ..., s)

注意：
1.A ≠ 0, B ≠ 0, AB可能等于0

    [0 1]

case1 A2x3 B3x4
case2 A3x2 B3x2

                    [1 1]     [-1 -1]
4.f(x) = anx^n + an-1x^n-1 + ... + a1x + a0 多项式
对于方阵有矩阵的多项式
f(A) = anA^n + an-1A^n-1 + ... + a1A + a0E

    [0 1 . 0]
    [. .   .]
    [0 0 . 1]

矩阵运算的性质
1.加法性质
A + B = B + A
(A + B) + C = A + (B + C)
2.乘法性质
kAB = (kA)B
(AB)C = A(BC)
3.混合性质
k(A + B) = kA + kB
A(B + C) = AB + AC

A = [a11 a12 ... a1n] X = [x1] b = [b1]
    [a21 a22 ... a2n]     [x2]     [b2]
    [... ...     ...]     [..]     [..]
    [am1 am2 ... amn]     [xn]     [bm]

AX = 0 齐次
AX = b 非齐

情形1
A是n阶矩阵，存在B，使得BA = E
有AX = b，可得BAX = Bb，则x = Bb
情形2
1.A是n阶矩阵，不存在B，使得BA = E
2.A是mxn矩阵，m ≠ n

可逆矩阵
设A是n阶矩阵，若存在n阶矩阵B，使得BA = E，
称A是可逆矩阵，B是A的逆矩阵，记做B = A^-1

伴随矩阵
A* = [A11 A21 ... An1]
     [A12 A22 ... An2]
     [... ...     ...]
     [A1n A2n ... Ann]
AA* = A*A = |A|E
A^-1 = (1 /|A|)A*
A可逆的充分必要条件是|A| ≠ 0

克莱姆法则
|A| ≠ 0，xi = |Ai| / |A|

矩阵分块
矩阵分块的背景：矩阵运算和矩阵应用过程中，若阶数高，为了方便计算，将矩阵进行分块
要求：加减时，A和B是同型矩阵且分块法一致
相乘时，A和B可乘且分的对应块要可乘

高斯消元法
2x1 - 2x2       + 6x4  = -2  -> x1 - x2       + 3x4 = -1
2x1 - x2  + 2x3 + 4x4  = -2          x2 + 2x3 - 2x4 = 0
3x1 - x2  + 4x3 + 4x4  = -3               x3  - 3x4 = -1
5x1 - 3x2 + x3  + 20x4 = -2                      x4 = 0
将线性方程组化成阶梯形线性方程组

数域F中m*n个数字aij(i = 1,2,...,m; j = 1,2,...n)排成m行n列，并括以圆括弧的数表称为数域F上的m*n的矩阵，通常用大写字母记做A或Am*n，有时记做A = (aij)m*n
[a11 a12 ... a1n]
[a21 a22 ... a2n]
[... ...     ...]
[am1 am2 ... amn]
当aij ∈ R，A为实矩阵
当aij ∈ C，A为复矩阵
当任意的aij都为0，记做0
当m=n时，A是n阶矩阵
系数矩阵A，增广矩阵(A,b)

当线性方程组的常数项b1 = b2 = ... = bm = 0时，我们称它为齐次线性方程组，否则称为非齐次线性方程组

有解的方程组称为相容方程组 无解的方程组称为不相容方程组

矩阵的加法 数量乘法 乘法
1）矩阵相等
如果两个矩阵A=(aij)和B=(bij)的行数和列数分别相等，且各对应元素也相等，即aij=bij，则称A和B相等，记作A=B
2）矩阵相加 当矩阵是同型矩阵，即行数和列数相同时，才可以相加
A + B = (aij + bij) = [a11 + b11 a12 + b12 ... a1n + b1n]
                      [a21 + b21 a22 + b22 ... a2n + b2n]
                      [...       ...           ...      ]
                      [am1 + bm1 am2 + bm2 ... amn + bmn]
3）矩阵的数量乘法
kA = (kaij) = [ka11 ka12 ... ka1n]
              [ka21 ka22 ... ka2n]
              [...  ...      ... ]
              [kam1 kam2 ... kamn]
4）矩阵的乘法 设A时一个m*n的矩阵，B时一个n*s的矩阵
A = [a11 a12 ... a1n] B = [b11 b12 ... b1s]
    [a21 a22 ... a2n]     [b21 b22 ... b2s]
    [... ...     ...]     [... ...     ...]
    [am1 am2 ... amn]     [bn1 bn2 ... bns]
C = AB = (cij) = ai1b1j + ai2b2j + ... ainbnj = (k=1 n)∑aikbkj

矩阵和行列式的区别
矩阵是m*n的数表
行列式是n*n的值
当D=|A|时，若D=0且A≠0，则A为奇异矩阵，若D≠0，则A为非奇异矩阵
