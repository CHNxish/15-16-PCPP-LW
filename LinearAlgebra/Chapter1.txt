行列式：线性代数中的一个工具，通过性质和基本定义求数值
重要的两个方法：数学归纳法 递推方法

二元线性方程组
a11x1 + a12x2 = b1
a21x1 + a22x2 = b2

x1 = D1 / D    x2 = D2 / D
D1 = |b1 a12|  D2 = |a11 b1|  D = |a11 a12|
     |b2 a22|       |a21 b2|      |a21 a22|

三元线性方程组
a11x1 + a12x2 + a13x3 = b1
a21x1 + a22x2 + a23x3 = b2
a31x1 + a32x2 + a33x3 = b3

x1 = D1 / D        x2 = D2 / D        x3 = D3 / D
D1 = |b1 a12 a13|  D2 = |a11 b1 a13|  D3 = |a11 a12 b1|  D = |a11 a12 a13|
     |b2 a22 a23|       |a21 b2 a23|       |a21 a22 b2|      |a21 a22 a23|
     |b3 a32 a33|       |a31 b3 a33|       |a31 a32 b3|      |a31 a32 a33|

n阶行列式
D = |a11 a12 ... a1n|
    |a21 a22 ... a2n|
    |... ...     ...|
    |an1 an2 ... ann|

D = a11A11 + a12A12 + ... + a1nA1n = (j=1 n)Σa1jA1j
Aij = (-1)^(i + j)Mij 
Aij代数余子式 Mij余子式

主对角线 次对角线 上三角行列式 下三角行列式
a11a22...a1n 相当于n = 1
(-1)^(n(n - 1) / 2) a1na2(n-1)...an1

行列式的性质 简化行列式的计算
性质1 行列式的行和列(按原顺序)互换，其值不变
性质2 行列式按任一行展开，其值相等
D = ai1Ai2 + ai2Ai2 + ... + ainAin = (j=1 n)ΣaijAij (i = 1,2,...,n)
性质3 行列式中某行或某列可以提出共同因子	行列式只改变一行或一列的情况下可以分称两个行列式
|a11  a12  ... a1n |    |a11 a12 ... a1n|
|...  ...      ... |    |... ...     ...|
|kai1 kai2 ... kain| = k|ai1 ai2 ... ain|
|...  ...      ... |    |... ...     ...|
|an1  an2  ... ann |    |an1 an2 ... ann|

|a11     a12     ... a1n    |   |a11 a12 ... a1n|   |a11 a12 ... a1n|
|...     ...         ...    |   |... ...     ...|   |... ...     ...|
|ai1+bi1 ai2+bi2 ... ain+bin| = |ai1 ai2 ... ain| = |bi1 bi2 ... bin|
|...     ...         ...    |   |... ...     ...|   |... ...     ...|
|an1     an2     ... ann    |   |an1 an2 ... ann|   |an1 an2 ... ann|
性质4 行列式中两行对应元素全相等，其值为零
性质5 在行列式中，把某行各元素分别乘非零常数k，再加到另一行的对应元素上，行列式的值不变
性质6 行列式的两行对换，行列式的值反号
性质7 行列式某行的元素乘另一行对应元素的代数余子式之和等于零
